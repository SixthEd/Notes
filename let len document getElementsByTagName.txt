	The fetch() function returns a Promise which is fulfilled with a Response object representing the server's response. You can then check the request status and extract the 	
	body of the response in various formats, including text and JSON, by calling the appropriate method on the response.

	or
	
	The res.json() method is used with the JavaScript Fetch API to parse the incoming response stream from a network request as JSON (JavaScript Object Notation) and convert it 
	into a usable JavaScript object. 

	How it Works
	When you use the fetch() method, it returns a Promise that resolves to a Response object. This initial Response object contains metadata like headers and status codes, but 
	the actual body of the response is a ReadableStream that has not yet been read or parsed. 
	The res.json() method is an asynchronous operation that: 
	Reads the entire response body stream to completion.
	Parses the body text as JSON.
	Returns a new Promise that, when resolved, provides the fully parsed JavaScript object. 
	Because res.json() is asynchronous, you must use .then() chaining or the await keyword to access the final data.
