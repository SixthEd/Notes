redis

Store data in ram in key value pair
It is live a one gaint json object
It is run on working memory of your computer called ram
It is mostly used for caching
Default port : 6379

redis-server: to check redis is working

redis-cli: to run the redis command after that add set key value pair
Ex: set name kyle
To get the value
Ex: get name  =  "kyle"

Whatever we are going to store in redis is going to be a type string by default
Ex set age 26
Ex get age = "26"

For delete
Ex del age
Ex get age = (nil)

To check key is exist
Ex Exists name 
true=1 false=0

Check to find all Keys using that match a certain pattern
Ex Keys * = all keys
1) "name"

To delete all keys
Ex flushall

Idea of expiration (ttl) time to live
Ex ttl name
if it is -1 then it has no expiration means it's going to forever

When can key expire 
Ex expire key seconds
   expire name 30
   now we do : ttl name it will give 30 and it will counting down
   at last it will give -2 means it is not exist
   
Add key expire when we set it
Ex: setex key seconds value
    setex name 10 kyle

for storing list

lpush key value

for get list
lrange key start stop
Ex lrange friends 0 -1  and -1 means get all the item in the list

for add end of the list : use rpush
